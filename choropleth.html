Choropleth Map

Choropleth map: a thematic map that shades enumeration units by their attribute values, primarily relying on colour value. This map allows us to visualize density changes in datasets. This map allows for a visual outline of the changing percentages of Key Biodiversity Areas (KBAs) that are protected terrestrial areas around the world. This is divided by enumeration from 0-100% in cover. This data pertains to the Indicator 15.1.2(b) of the SDG Goal #15. The proportion of important sites for terrestrial biodiversity that are covered by protected areas show temporal trends in the mean percentage of each important site for terrestrial biodiversity (i.e., those that contribute significantly to the global persistence of biodiversity) that is covered by designated protected areas. This was the first map I created, which was learning an entire skill. I had done some data processing before in other classes but this type took adjusting the data to be able to fit the map template, and adjusting the colors to be purposeful. I found myself thinking further on how the data needed to be presented to have the user see differences in the data. I chose to have a gradient from yellow-green shades in lower percentage values to darker green in the highest percentages of protected areas. I wanted an association with lush vegetation at high protection.
